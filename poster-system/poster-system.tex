\section{Component Lifetimes}
The lifetime for each \comp{k} is assumed as Weibull with fixed shape $\kappa_k$:
$\displaystyle F_k(t \mid \lambda_k) = 1 - e^{-\frac{t^{\kappa_k}}{\lambda_k}}$.
Then, $\E[T\mid\lambda_k] = \sqrt[\kappa]{\lambda_k} \Gamma(1+1/\kappa)$.
%
%For each component type,
We have information on $\lambda_k$ from the component manufacturer,
but do not fully trust it and model knowledge on $\lambda_k$ cautiously with a \emph{set of priors} $\MkZ$.

\section{Set of Priors}
Each $\MkZ$ is taken as a set of conjugate inverse Gamma priors in canonical parametrization \cite{diss,Walter2009a}:
$\MkZ = \big\{ \ig(\nkz+1, \nkz\ykz) \mid [\nkzl,\nkzu] \times [\ykzl,\ykzu] \big\}$,
where $\ykz = \E[\lambda_k\mid\nkz,\ykz]$ and $\nkz =$ pseudocounts or prior strength
